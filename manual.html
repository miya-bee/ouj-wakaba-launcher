放送大学 WAKABA ランチャー 取扱説明書

公開版 V1_0_0

この拡張でできること

  • WAKABAトップへすぐ戻る
  • 配信トップへすぐ移動する
  • 登録科目をランチャーから開く
  • 科目履歴・個別履歴から再開する
  • ログアウト状態から自動ログインを補助する

最初の設定

  1. 拡張アイコンを押して 管理 を開きます。
  2. 必要なら ログインID と パスワード を入力します。
  3. 必要に応じて 科目履歴の表示数 と 個別履歴の表示数 を設定します。
  4. 保存 を押します。

保存先はこのPCの Chrome 内だけです。共有PCでは使わないでください。

基本的な使い方

WAKABAトップを開く: popup の WAKABAトップ を押します。

配信トップを開く: popup の 配信トップ を押します。

今の科目ページを登録: 科目ページ を開いた状態で popup の 今の科目ページを登録 を押します。

手動登録: 管理画面の 科目登録 で科目名と URL を入力して保存します。

履歴から開く: popup の 個別履歴 または 科目履歴 を押します。

バックアップ

エクスポート: 管理画面で エクスポート を押すと JSON で保存できます。

インポート: 管理画面で インポート を押して JSON を読み込みます。

注意

  • 個別回ページは登録対象外です。
  • サイト構成が変わると、履歴や科目名取得の動作が変わることがあります。
  • 共有PCではログイン情報を保存しないでください。
  • 外部送信する処理は入れていません。
