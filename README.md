# 放送大学 WAKABA ランチャー

公開版 **V1_0_1** です。  
放送大学の **WAKABA** と **配信ページ** を開きやすくする Chrome 拡張です。

WAKABAトップへ戻る固定メニュー、科目ページの登録、科目履歴・個別履歴の表示、自動ログイン補助をひとつにまとめています。

## バージョン

- 公開版表記: **V1_0_1**
- Chrome 拡張の内部バージョン: **1.0.1**

## 主な機能

- **WAKABAトップ** をワンクリックで開く
- **配信トップ** をワンクリックで開く
- **登録科目** をランチャーからすぐ開く
- **科目履歴** から最近見た科目を開く
- **個別履歴** から最近開いた講義回を開く
- **今の科目ページを登録** で科目ページを登録する
- ログアウト状態からの **自動ログイン補助**
- 設定の **エクスポート / インポート**
- 設定画面の **使い方** ボタンから取扱説明書を開く
- **科目履歴の表示数** と **個別履歴の表示数** を別々に設定できる

## 想定する使い方

- WAKABAトップへ素早く戻りたい
- よく使う科目ページをランチャーに登録したい
- 最近見ていた科目や講義回をすぐ開きたい
- ログアウト時のログイン操作を少しでも減らしたい

## 導入方法

1. このリポジトリをダウンロードするか、Release に添付した zip を取得します。
2. zip の場合は展開します。
3. Chrome で `chrome://extensions/` を開きます。
4. 右上の **デベロッパー モード** を ON にします。
5. **パッケージ化されていない拡張機能を読み込む** を押します。
6. このフォルダを選びます。

## 最初の設定

1. 拡張アイコンを押して **管理** を開きます。
2. 必要なら **ログインID** と **パスワード** を入力します。
3. 必要に応じて
   - **科目履歴の表示数**
   - **個別履歴の表示数**
   を設定します。
4. **保存** を押します。

## 基本的な使い方

### WAKABAトップを開く

- 拡張アイコンを押します。
- **WAKABAトップ** を押します。

### 配信トップを開く

- 拡張アイコンを押します。
- **配信トップ** を押します。

### 今見ている科目ページを登録する

- 配信授業の **科目ページ** または WAKABA の科目ページを開きます。
- 拡張アイコンを押します。
- **今の科目ページを登録** を押します。
- 必要に応じて **管理** 画面で科目名や URL を確認します。

※ **個別回ページは登録対象外** です。

### 登録科目を手動で追加する

- 拡張アイコンを押して **管理** を開きます。
- **科目名** と **科目URL** を入力します。
- **科目を保存** を押します。

### 設定をバックアップする

- **管理** を開きます。
- **エクスポート** を押します。
- JSON ファイルが保存されます。

### 設定を復元する

- **管理** を開きます。
- **インポート** を押して JSON ファイルを選びます。

### 取扱説明書を開く

- **管理** を開きます。
- **使い方** を押します。

## プロジェクト構成

- `manifest.json` — 拡張の定義ファイル
- `background.js` — 起動制御、履歴保存、設定保存
- `content.js` — ページ上の情報取得、自動ログイン補助
- `popup.*` — ランチャー画面
- `options.*` — 管理画面
- `manual.html` — 取扱説明書
- `icons/` — アイコン

## データとプライバシー

- ログイン情報の保存先は、この PC の Chrome 内の `chrome.storage.local` のみです。
- 個人情報や認証情報を外部送信する処理は入れていません。
- 共有PCではログイン情報を保存しないでください。

## 注意

- サイト構成が変わると、履歴や科目名取得の動作に影響することがあります。
- 個別回ページは登録対象外です。
- Chrome の仕様変更や CSP 変更により、自動ログイン補助の動作が変わることがあります。
