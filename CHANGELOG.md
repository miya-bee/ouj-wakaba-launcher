# Changelog

## V1_0_2

### 追加・変更
- 固定メニューを 4 項目化し、**WAKABAトップ / 配信トップ / Gmail / シラバス** を表示
- 旧 **「管理」** 表記を **「設定」** に変更
- シラバスの既定URLを、トークン付きURLから公開のシラバス検索ページへ変更
- popup の縦方向を詰め、4項目でも見やすいレイアウトに調整
- popup 全体の横幅と余白を見直し、ランチャー枠の右側が切れて見えにくい問題を調整
- **個別履歴 / 科目履歴 / 登録科目** の囲み枠色を見分けやすく変更

### 修正
- 配信トップボタンで、ログイン状態を確認してから目的ページへ送るよう起動処理を調整し、ジャンプの安定性を改善
- 保存済みの旧シラバスURLが残っている場合でも、公開URLへ置き換えるよう修正
- 科目トップページで **「今の科目ページを登録」** を押した際に、直前の個別回操作の影響で誤って登録不可になることがある問題を修正
- 個別回ページを登録不可とする仕様は維持
- 長い科目名は右端まではみ出さないよう、省略表示に統一
